# FreeVPN by C3ph3i

VPN client using VPN access from http://vpnbook.com/

# Requirements
  - PHP
  - UNIX based OS
  - Composer

# Installing and run

```sh
$ git clone https://github.com/alkhachatryan/freevpn.git
$ cd freevpn
$ composer install
```

Once you installed the program - run it.

```sh
$ sudo  ./run.sh
```
